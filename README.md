## About the project
Rave chat is a minimal chat application with basic peer to peer messaging (Just another portfolio projcet ðŸ™„).

Get it on [playstore](https://play.google.com/store/apps/details?id=com.abumuhab.chat) and play around with it 

## Built with
- Nodejs
- Firebase
- Kotlin
- Socket.io

## Features
- Social signup (Snapchat)
- Snapchat Bitmoji AVI
- Real time chat
- Chat Notifications

## Future updates
- Voice mesages
- Image and Video sharing
- Group chats

## Backend
