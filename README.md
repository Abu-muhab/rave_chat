## About the project
Rave chat is a minimal chat application with basic peer to peer messaging. 

### Built with
- Nodejs
- Firebase
- Kotlin
- Socket.io

## Features
- Social signup (Snapchat)
- Snapchat Bitmoji AVI
- Real time chat
- Chat Notifications
